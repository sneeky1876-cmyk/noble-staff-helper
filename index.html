How it works

Built around the NoblePrac workflow

The extension reads session team data, stores a local Epic-name to Discord-ID cache, and adds actions only on NoblePrac session and leaderboard pages. Creator checks use Fortnite-API; preferences and cached IDs remain in Chrome storage.
